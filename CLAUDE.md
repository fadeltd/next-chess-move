# Next Chess Move

A modern chess analysis tool powered by Stockfish with interactive board, real-time move analysis, and chess variations exploration.

## Project Guidelines

- Built with Next.js 16 and TypeScript
- Uses chess.js for move validation and Stockfish.js for engine analysis
- Responsive design with mobile and desktop support
- Type-safe code with proper TypeScript interfaces

See [AGENTS.md](AGENTS.md) for Next.js-specific guidelines.
